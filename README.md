# zhinen-php

# 要求：

* PHP 5.4.0+

## 目录说明

```
project                 根目录
├─app                   应用目录
│  ├─controllers        控制器目录
│  ├─models             模块目录
│  ├─views              视图目录
├─config                配置文件目录
├─fastphp                核心目录
├─static                静态文件目录
├─index.php             入口文件
```

## 使用

然后访问站点域名：http://localhost/ 就可以了。
